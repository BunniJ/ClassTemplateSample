# ClassTemplateSample

Implement a Generic Array class, which needs to support:

1. Storage of data for built-in data and custom datatypes.
2. Store the data in the array into the heap area.
3. The capacity of the array can be passed into the constructor.
4. Provide the corresponding copy constructor and operator= to prevent shallow copy issue.
5. Provide tail insertion and tail deletion functions to allow user add or delete elements in an array.
6. Elements in an array can be accessed by its index.
7. Get the capacity and size of an array.
